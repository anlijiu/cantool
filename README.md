# cantool

send/receive CAN/CANFD on linux

### support device

+ usbcanii
+ tsmaster

> tsmaster 目前需要 sudo 执行才可以打开设备。 并且必须用 sudo ./cantool 打开， 貌似libTSCANApiOnLinux.so 会到主程序所在目录dlopen libTSH.so
